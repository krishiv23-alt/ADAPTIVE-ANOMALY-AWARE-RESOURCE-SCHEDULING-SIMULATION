# ADAPTIVE-ANOMALY-AWARE-RESOURCE-SCHEDULING-SIMULATION

This project simulates a Kubernetes-like scheduler with:

- Pod monitoring
- Anomaly detection
- Load balancing
- Self-healing
- Web dashboard visualization

## Tech Stack
- Python (Flask)
- HTML, CSS, JavaScript
- Chart.js

## How to Run

pip install flask  
python app.py  

Open http://127.0.0.1:5000
